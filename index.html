Image-Pro

The Image Pro Platform is a versatile web application designed to empower users with AI-driven image creation and manipulation. It seamlessly integrates two core functionalities: an Image Generator that transforms textual prompts into unique visuals using advanced generative AI, and an Image Editor that allows users to upload their own images and apply AI-powered transformations based on descriptive text inputs. The platform features a responsive and intuitive user interface, ensuring a smooth experience across various devices, and includes a persistent gallery to store and manage all generated and edited images, offering options for viewing, downloading in multiple formats, and deleting.

Make My Quiz

This "Make My Quiz" application is a dynamic web-based tool enabling users to effortlessly create personalized quizzes by inputting a topic or uploading an image, alongside specifying the number of questions, difficulty level, and answer format (multiple-choice, direct answer, or true/false). It offers an engaging interactive experience with immediate feedback on answers, a visual progress bar to track advancement, and optional sound effects for a more immersive feel, all while leveraging a large language model to provide on-demand explanations for correct answers and fun facts related to the quiz topic. Furthermore, users can review their completed quizzes and conveniently export either the quiz questions or a detailed review, including their answers and the correct ones, into a printable PDF format, all within a responsive design that ensures optimal usability across various devices.

AI Chatbot

An interactive chat interface where you can type messages and receive responses from an AI. It offers the ability to upload images directly from your device for multimodal conversations, allowing the AI to understand and respond to visual input along with text. Features include a "New Chat" option to start fresh conversations and a "History" sidebar to easily navigate and revisit past chat sessions, which are saved directly in your browser. The application boasts a responsive design, ensuring optimal functionality across various devices, and provides clear visual feedback when the AI is processing requests or an image is being uploaded.
